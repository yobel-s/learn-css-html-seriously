Web Technologies

Web technologies are the tools and standards used to create and manage web content.

HTML
Hypertext Markup Language is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.
CSS
Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.
JavaScript
JavaScript is a high-level, often just-in-time compiled, and multi-paradigm programming language. It has become a standard for creating interactive web pages and is an essential part of web applications.
